A modified version of the space-themed two-step task with
two ships, three plantes, six aliens. one of the ships deterministically 
leads to the first planet. the other ship's transition is evenly
split between the second and the third planet. 
Alien rewards are consisten but noisy.

Assumes that the lib folder from nivturk with jspsych packages is 
already in the static folder. 


adapted from Sam Zorowitz, Gili Karni, Branson Byers
